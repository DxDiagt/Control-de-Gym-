# Happy Body Gym - Sistema de Control de Asistencia

Sistema de gestión para el control de asistencia, pagos y observaciones de miembros del gimnasio Happy Body Gym.

## Requisitos del Sistema

- Windows 10 o superior
- Microsoft Access Database Engine 2010 Redistributable o superior
- 4GB RAM mínimo recomendado
- 500MB espacio en disco

## Funcionalidades

- Gestión de miembros
- Control de asistencia
- Registro de pagos (mensual/quincenal)
- Sistema de observaciones
- Seguimiento de pagos vencidos
- Interfaz moderna y fácil de usar

## Instalación

1. Ejecute el instalador HappyBodyGym_Setup_v1.1.exe
2. Siga las instrucciones del asistente de instalación
3. El programa creará accesos directos en el escritorio y menú inicio
